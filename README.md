# Level & Gradient Calculator

A simple web-based tool for calculating levels, gradients, and distances for landscape and architectural design.

## 🌐 Live Demo
https://qizeng-design.github.io/Level-calculator/

## ✨ Features

- Input any **three of four parameters**:
  - High level (m)
  - Low level (m)
  - Distance (mm)
  - Gradient (e.g. 1:21)
- Automatically calculates the missing value
- Outputs **height difference**
- Generates a **section diagram** to visualise the slope
- Diagram reflects **true slope proportions**, scaled to fit the screen

## 📐 Example

- High level: 64.145 m  
- Gradient: 1:21  
- Distance: 15,212 mm  

→ The tool calculates the **low level** and displays the slope visually.

## 🛠️ How to Use

1. Enter any three values
2. Click **Calculate**
3. View the result and diagram

## 💡 Purpose

This tool is designed to:
- Speed up level and gradient calculations
- Reduce manual errors
- Provide quick visual understanding of slopes
- Support landscape architects and designers in early-stage design
